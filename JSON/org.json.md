###JSON的优势
JSON(JavaScript Object Notation) 是一种**轻量级**的数据交换格式，它的**可读性**、**可扩展性**、**简易程度**都比XML要好很多，所以现在已经成为主流的数据传输格式 。

###JSON的劣势
  1. 没有日期类型
  2. 内联没法注释
  3. JSON字段过长时，其格式就会显得非常凌乱


###JSON在JAVA的简单使用

  1. 在pom.xml添加JSON的依赖 

   	 	<dependency>
			<groupId>org.json</groupId>
	    	<artifactId>json</artifactId>
	    	<version>20200518</version>
		</dependency>
